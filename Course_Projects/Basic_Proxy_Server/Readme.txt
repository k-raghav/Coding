				Readme

1. For Compiling the codes run make
2. That creates the execulables client and proxy
3. The executable proxy serves as the proxy server and the executable client serves as client
4. You can run the executable by using ./proxy.
5. A client can establish a connection with the opened proxy by using ./proxy.
6. Then the client starts execution and asks for typing in something.
7. You can select to either query about the urls for a domain name or typein an url and look
for the html page.
8. If you select to retreive a page then you are allowed to typein the url.
9. After you type in the url you are requested to select conditional get or non - cond - get.
10. Based on type or request and the type of get request the request is served by server program.
